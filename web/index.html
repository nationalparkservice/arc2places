arc2places

Manage Places data with ArcGIS

About arc2places

arc2places is a suite of tools for uploading National Park Service GIS data in ESRI data format to the NPS Places system. It was built to support the centennial data call.

arc2places provides an ArcGIS Toolbox interface for using these tools. This document describes how to install and use the arc2places toolbox. Source code and additional information on the arc2places tools can be found on GitHub.

Installing

Assumptions

  • You have ArcGIS 10.1sp1 or greater installed on your computer, with python installed in the default location.
  • You have administrative permissions on your computer (at least write permission to the python folder).
  • That windows knows how to open python files (*.py) with the python interpreter (typical with ArcGIS install).
  • If you have pip installed, it is a current, working installation.
  • You do not have multiple versions of python installed.

Instructions

  1. Download arc2places. If the clicking the link does not download the zip file, then right click on the link and select 'Save link as...'
  2. Unzip the download and put it in a location easily accessible to ArcCatalog.
  3. Use windows explorer to browse to the exraction location of arc2places.zip.
  4. If you are feeling lucky
    • Double click on install.py
  5. Otherwise:
    1. Right click on install.py and select Edit with IDLE.
    2. In the IDLE Editor, select Run -> Run Module from the menu (or press F5).
    3. This should open another window that runs the installation program and displays diagnostic output.
    4. If you see error messages in the IDLE window, send them to regan_sarwas@nps.gov for installation support.
    5. Close both IDLE windows
  6. Open ArcCatalog and browse to the extraction location of arc2places
  7. Try and expand the arc2places toolbox
    • If it opens and you see the contents of the toolbox, installation was successful.
    • If there is a red x on the toolbox icon, or it does not expand, then contact regan_sarwas@nps.gov

Using

The toolbox can be used in ArcCatalog and ArcMap. The documentation below is based on ArcCatalog. The toolbox has only been tested on ArcGIS 10.3. It should work on ArcGIS 10.1 sp1 or higher. It does not yet work with ArcGIS Pro.

Testing

Before uploading any production data, you are encouraged to test with small datasets. Testing should be done with the test version of the Places database. The tools have a checkbox to select publishing to test vs. production. View the test version of Places at http://insidemaps.nps.gov/dist_dev/index.html. You should verify that sensitive features were not sent, and that the items got the correct tags/symbology.

During testing you are also encouraged to use the tools under 'Seed Places Step by Step'. The 2nd and 3rd tool are optional based on the results of the validation check. Step 6 is also not required. Steps 1,4 and 5 (and optionally 6) are combined into a single tool called 'See Places with eGIS Data'

Step by Step Upload

Uploading step by step gives you more information in what is happening in the upload processing, and allows you to respond more easily if something goes wrong.
